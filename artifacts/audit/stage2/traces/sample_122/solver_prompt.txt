Problem Statement:
### Problem Statement

In the context of a next-generation smart city infrastructure project, an engineering team is tasked with designing a multi-layer electrical distribution network that must simultaneously satisfy precise capacity constraints, demand-supply balancing, and resiliency requirements. This is formulated as a **minimum-cost maximum-flow problem** on a multi-layer network.

#### Network Structure
The city layout is represented as a set of interlinked graphs (one per layer). Nodes can appear in multiple layers, each with a distinct demand, supply, or throughput requirement:
- **Supply nodes** (positive values): Energy producers
- **Demand nodes** (negative values): Energy consumers  
- **Relay nodes** (zero net balance): Intermediate connection points

Edges represent potential connections with two key attributes:
- **Cost**: Installation/operational cost per unit
- **Capacity**: Maximum flow throughput

#### Optimization Model

This is a **minimum-cost flow problem** with the following formal definition:

**Decision Variables:**
- Let f_{e} denote the flow on edge e

**Objective Function:**
Minimize: Σ (adjusted_cost(e) × f_{e}) for all edges e with flow

Where adjusted_cost(e) is computed as:
```
adjusted_cost(e) = base_cost(e) × priority_factor(e)

priority_factor(e) = 1.0 / (priority_weight(u) × priority_weight(v))
  where u, v are the endpoints of edge e
  and priority_weight(node) = max(0.1, critical_nodes.get(node, 1.0))
```

**Higher priority values → Lower effective costs → Preferential selection**

**Constraints:**
1. **Flow Conservation**: Σ flow_in(v) - Σ flow_out(v) = balance(v) for all nodes v
2. **Capacity Constraints**: 0 ≤ f_{e} ≤ capacity(e) for all edges e
3. **Supply-Demand Balance**: Σ supply = Σ |demand| (globally)
4. **Critical Node Connectivity**: All critical nodes with non-zero balance must be connected to at least one supply node in the solution network

**Feasibility Requirement:**
If no feasible solution exists (e.g., supply ≠ demand, insufficient capacity, or critical nodes cannot be connected), return an empty list [].
