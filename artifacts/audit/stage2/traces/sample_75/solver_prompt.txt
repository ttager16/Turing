## Problem Statement

The task is to manage a hospital’s **emergency department queue**.
Each patient has a **unique ID** and an **initial severity score**, which dynamically changes after each treatment.
The goal is to produce a **treatment order** that updates severities after each patient is treated.

### Rules

* **Treatment Priority:**
  Patients with higher severity are treated first.

* **Aging Adjustment:**
  After each patient is treated, the remaining patients’ severities are updated as follows:

  * Odd patient IDs -> severity + `aging_interval`
  * Even patient IDs -> severity - `aging_interval`

* **Arrival Order (Tie-Breaker):**
  If two patients have the same severity, the one who appeared earlier in the input list is treated first.

* The aging adjustment happens **after** each treatment, not before the first one.
