Problem Statement:
A financial data analysis company monitors large datasets of transaction descriptions to detect potential fraudulent activities.
Fraudsters often hide their manipulations by embedding mirrored or palindromic patterns within transaction notes — for example, `"AB1221BA"` or `"REVEREVER"`.

To strengthen their fraud detection system, the company wants to develop a Python function that can:

1. Identify the **longest palindromic substring** in a given transaction description.
2. Support **range-based Least Common Multiple (LCM)** queries over the lengths of all detected palindromic substrings using a segment tree.

This enables the system to efficiently identify suspicious mirrored patterns in text data that could indicate fraud or manipulation.



## Input

* `notes` — a string representing the transaction description.
* `queries` — a list of list `(L :`int`, R: `int`)` where `L` and `R` representing index ranges in the palindrome length list. Each element in list is unique
  For each query, compute the **LCM** of all palindrome lengths between indices `L` and `R` inclusive.



## Output

Return a dictionary with the following keys:

* `"longest_palindrome"` -> the longest palindromic substring found in the transaction description.
* `"lengths"` -> a list of lengths of all palindromic substrings detected.
* `"lcm_results"` -> a list containing the LCM results for each `(L, R)` query.

