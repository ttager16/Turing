Problem Statement:
## Objective

Design a program to calculate the Dynamic Cooling Efficiency Index (DCEI) for an industrial refrigeration system using a sequence of time-ordered sensor readings.

For each contiguous subarray (window) of a given size, compute the DCEI as:

```
DCEI = (maximum temperature - minimum temperature) / (maximum power - minimum power)
```

The computation must be implemented efficiently using the **monotonic queue** technique to ensure linear time performance. The program should validate all inputs carefully and return a clear error object if any invalid condition occurs.

```python

```

**Parameters**

* `temperature_readings`: A list of integers representing temperature readings in degrees Celsius.
* `power_readings`: A list of integers representing power usage readings in kilowatts.
* `window_size`: A positive integer indicating the size of the sliding window.

**Return Value**
The function should return a dictionary with one of the following keys:

* `'result'`: Contains a list of floating-point DCEI values, each rounded to two decimal places.
* `'error'`: Contains a string describing the first validation error encountered.

## Problem Description

The refrigeration unit produces time-ordered logs of temperature and power readings. For a chosen window size, you must analyze all possible contiguous subarrays of that length and compute the DCEI for each.

To ensure performance, you must use **four monotonic queues (deques)**:

* `temp_max`: stores indices in decreasing temperature order (front is maximum).
* `temp_min`: stores indices in increasing temperature order (front is minimum).
* `power_max`: stores indices in decreasing power order (front is maximum).
* `power_min`: stores indices in increasing power order (front is minimum).

At each step:

1. Remove indices from the front of each deque that are outside the current window.
2. Remove elements from the back while maintaining monotonic order.
3. After forming a complete window, compute:

   * `temp_fluct = max_temperature - min_temperature`
   * `power_delta = max_power - min_power`
4. If either of these values violates the constraints below, return an appropriate error object.
5. Otherwise, compute `dcei = round(temp_fluct / power_delta, 2)` and store it in the results list.
