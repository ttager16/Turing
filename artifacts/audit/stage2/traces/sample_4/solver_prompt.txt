# Polynomial Multiplication using Fast Fourier Transform

## Problem Statement

Implement an efficient polynomial multiplication function using the Fast Fourier Transform (FFT) algorithm. Given two polynomials represented as lists of integer coefficients, compute their product polynomial.

Polynomials are represented with the constant term first. For example, `[2, 0, 1, 3]` represents the polynomial `2 + 0x + 1x² + 3x³` or simply `2 + x² + 3x³`.

## Objectives

1. **Implement FFT-based Multiplication**  
   Use the Fast Fourier Transform (FFT) algorithm to efficiently multiply two polynomials.

2. **Handle Integer Coefficients**  
   Work with integer coefficients in the range [-10⁶, 10⁶]. FFT operates on complex numbers, so results must be rounded back to integers with proper precision handling.

3. **Comprehensive Edge Case Handling**  
   The solution must correctly handle various edge cases including zero polynomials, single coefficients, negative values, and large inputs.

## Constraints

- Polynomials can have up to **10⁵ coefficients**
- Coefficients range from **-10⁶ to 10⁶**
- No third-party libraries (e.g., `numpy`, `scipy`) allowed

## Edge Cases to Handle

1. **Zero Polynomials**: Product of any polynomial with `[0]` or `[]` is `[0]`
2. **Single Coefficients**: `[k] × poly` should efficiently compute scalar multiplication
3. **Negative Coefficients**: Correctly handle sign propagation.
4. **Trailing Zeros**: Trim trailing zeros from the result.
5. **Empty Inputs**: Treat `[]` as `[0]`
6. **Different Sizes**: Handle polynomials with significantly different degrees
7. **Floating-Point Precision**: Round FFT results carefully to avoid precision errors
8. **Large Coefficients**: Product coefficients may exceed the input range.
9. **Internal Zero Coefficients**: Handle polynomials where multiplication produces exact zeros in the middle of the result. These internal zeros must be preserved and not trimmed like trailing zeros. FFT rounding must produce exact integer zeros, not near-zero values like `0.0000001`
